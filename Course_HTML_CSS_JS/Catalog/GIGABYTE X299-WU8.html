Подробное описание товара

GIGABYTE пополнила системная плата X299-WU8 типоразмера CEB (305 х 267 мм). Она построена на чипсете Intel X299 Express и рассчитана на процессоры Intel Core X в исполнении LGA2066. Важной особенностью платы является наличие семи слотов расширения PCIe 3.0 x16. Два моста PCIe x48 PLX PEX8747, преобразующие две линии PCIe 3.0 x16, предоставленные процессором, позволяют получать конфигурации слотов x16 / NC / x16 / NC / x16 / NC / x16 и x16 / x8 / x8 / x8 / x8 / x8 / x8. Другими словами, один слот всегда работает в режиме x16, в то время как остальные шесть делят между собой три линии x16 в зависимости от заполнения.
